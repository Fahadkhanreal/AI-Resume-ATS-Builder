# Feedback: Constitution as Source of Truth

**Rule:** Constitution is the authoritative source of truth for all project decisions. No decision can contradict the 8 immutable core principles.

**Why:** Fahad established explicit governance model where principles are non-negotiable. This ensures consistency, quality, and alignment across all development phases (spec, plan, tasks, implementation).

**How to apply:**
- Before proposing any architectural decision, verify alignment with all 8 core principles
- If a decision conflicts with a principle, surface the conflict and request explicit amendment
- All PRs, specs, and plans must reference applicable principles
- Amendment procedure: user approval required; version bump follows semantic versioning
- When in doubt, defer to the Constitution over convenience or speed
